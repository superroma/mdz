# Руководство по форматированию Markdown (c расширениями GFM)

Ниже для каждого элемента показано **как он выглядит** и **как его написать в Markdown**.

---




## Абзацы и перенос строки

Обычный текст формирует абзац.

Строка с мягким переносом в конце␣␣
Продолжение строки.

```md
Обычный текст формирует абзац.

Строка с мягким переносом в конце␣␣  
Продолжение строки.
```

*(␣␣ означает два пробела в конце строки)*

## Маркированный список

* Первый пункт
* Второй пункт

  * Вложенный пункт

```md
- Первый пункт
- Второй пункт
  - Вложенный пункт
```

## Нумерованный список

1. Первый пункт
2. Второй пункт
3. Третий пункт

```md
1. Первый пункт
2. Второй пункт
3. Третий пункт
```

| текст          |   текст   |           текст |
| ещё текст      | ещё текст |       ещё текст |

```md
| По левому краю | По центру | По правому краю |
|:---------------|:---------:|----------------:|
| текст          | текст     | текст           |
| ещё текст      | ещё текст | ещё текст       |
```

## Комбинированный пример (несколько возможностей сразу)

| Задача        | Статус |
| ------------- | :----: |

| Вторая задача |    ⏳   |

```md
Таблица статусов:

| Задача        | Статус  |
|---------------|:-------:|
| Первая задача |   ✅    |
| Вторая задача |   ⏳    |
```
